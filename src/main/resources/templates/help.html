Help & Documentation

Getting Started

Welcome to the Construction Daily Reporting System. This guide will help you get started with the application.

Note: If you're new to the system, we recommend going through this guide step by step.

System Overview

The Construction Daily Reporting System helps construction teams document daily activities, track progress, and monitor weather conditions that may affect site operations. The system features:

  • Daily Reports: Create and manage reports of daily construction activities
  • Weather Services: Access current and historical weather data for construction sites
  • User Management: Different roles and permissions for team members
  • API Access: Integrate with other systems via REST, GraphQL, SOAP, or gRPC

Quick Start Guide

  1. Log in with your provided credentials
  2. Navigate to the Daily Reports section from the main menu
  3. Click "Create New Report" to start documenting daily activities
  4. Fill in the required information and add activities as needed
  5. Submit the report for approval when complete

Daily Reports

Creating a New Report

To create a new daily report, navigate to the Reports section and click on the "Create New Report" button. You'll need to provide the following information:

  • Project ID: The identifier for the construction project
  • Report Date: The date for which the report is being created
  • Notes: Any general notes about the day's activities or conditions

Adding Activities

After creating a report, you can add individual activities by clicking the "Add Activity" button. For each activity, provide:

  • Category: The type of activity (e.g., Excavation, Concrete, Electrical)
  • Description: Details about the activity
  • Start/End Time: When the activity began and ended
  • Progress: The percentage of completion for the activity
  • Status: The current state (Not Started, In Progress, Completed, Blocked)
  • Personnel: Team members involved in the activity

Report Workflow

Reports go through the following status transitions:

  1. DRAFT: Initial state, can be edited
  2. SUBMITTED: Sent for review, awaiting approval
  3. APPROVED: Finalized and cannot be modified
  4. REJECTED: Returned for corrections
Important: Once a report is approved, it cannot be modified. Ensure all information is accurate before submitting.

Weather Services

Current Weather

To check current weather conditions for a construction site, navigate to the Weather section and select "Current Weather." Enter either a city name or zip code to retrieve the current conditions.

Historical Weather Data

The system allows you to retrieve weather data for past dates, which can be useful for documentation and validation purposes. To access historical data:

  1. Navigate to the Weather section and select "Historical Weather"
  2. Enter the location information (city or zip code)
  3. Specify the date range (limited to 7 days in the past)
  4. Submit the request to view historical conditions

Weather Alerts

The Weather Alerts feature analyzes current conditions and identifies potential hazards that might affect construction activities. The system checks for:

  • Extreme temperatures (high heat or freezing conditions)
  • High winds that may affect crane operations
  • Precipitation that could impact excavation or concrete work
  • Lightning hazards
  • Poor visibility conditions

Temperature Conversion

The Temperature Conversion tool allows you to convert between Fahrenheit and Celsius, which can be helpful when working with material specifications or international standards.
